# 巡检/报表

## GET 统计报表：完成率、异常数量、人员工作量、异常分布、高风险设备。

GET /inspection/report

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|period|query|string| 是 |none|

### 说明（与 Apifox / OpenAPI 一致）

- **正式响应数据模型**：`InspectionReportResponseVO`（见下方第二个 JSON 示例），字段包括 `period`、`periodTitle`、`statisticsRangeDescription`、`overview`、`trendSeries`、`staffWorkload`、`exceptionStatusDistribution`、`highRiskDevices` 等。
- **下方第一个 JSON 示例**（含 `stats`、`trend`、`personWorkload`、`exceptionDistribution`、`riskDevices`）对应历史/演示用的 `InspectionReportVO` 形态，**不应作为当前接口对接依据**。若后端仍返回 `riskDevices`，前端会做兼容映射为 `highRiskDevices` 展示结构；新接入请按 `InspectionReportResponseVO` 实现。

<h2 id="tocS_InspectionReportOverviewVO">InspectionReportOverviewVO</h2>

```json
{
  "inspectionCompletionRatePercent": 0,
  "inspectionCompletionRateCompareNote": "string",
  "newExceptionCount": 0,
  "newExceptionCountDeltaVsPreviousPeriod": 0,
  "newExceptionCountCompareNote": "string"
}

```

### 属性

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|inspectionCompletionRatePercent|number|false|none||巡检完成率（0～100，百分比数值）|
|inspectionCompletionRateCompareNote|string|false|none||与上一可比周期的完成率对比说明，如「较上期 +1.2%」|
|newExceptionCount|integer(int64)|false|none||本周期内新建异常单数量（按异常单 createTime 统计）|
|newExceptionCountDeltaVsPreviousPeriod|integer(int64)|false|none||较上一可比周期异常数量变化（正数表示增多）|
|newExceptionCountCompareNote|string|false|none||与上期异常数对比说明，如「较上期 +3 条」|
