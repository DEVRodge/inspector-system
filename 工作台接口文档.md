# 工作台

## GET 工作台统计（设备总数、今日/本月任务与完成率来自巡检记录，待处理/待指派来自异常）

GET /inspection/dashboard/stats

> 返回示例

> 200 Response

```json
{
  "totalDevices": 0,
  "devicesAddedThisMonth": 0,
  "todayTaskTotal": 0,
  "todayTaskCompleted": 0,
  "todayCompletionRate": 0,
  "monthTaskTotal": 0,
  "monthTaskCompleted": 0,
  "monthCompletionRate": 0,
  "pendingExceptionCount": 0,
  "unassignedExceptionCount": 0
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[InspectionDashboardStatsVO](#schemainspectiondashboardstatsvo)|

### 属性

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|totalDevices|integer(int64)|false|none||设备总数（设备台账总数）|
|devicesAddedThisMonth|integer(int64)|false|none||较上月增加台数（本月新增设备数）|
|todayTaskTotal|integer(int64)|false|none||今日应执行任务数（当日 executeTime 的巡检记录数）|
|todayTaskCompleted|integer(int64)|false|none||今日已完成任务数（当日应执行且 status=COMPLETED）|
|todayCompletionRate|number|false|none||总体完成进度（今日已完成/今日应执行×100），0-100|
|monthTaskTotal|integer(int64)|false|none||本月应执行任务数（executeTime 在本月内的巡检记录数）|
|monthTaskCompleted|integer(int64)|false|none||本月已完成任务数（本月应执行且 status=COMPLETED）|
|monthCompletionRate|number|false|none||本月完成率（本月已完成/本月应执行×100），0-100|
|pendingExceptionCount|integer(int64)|false|none||待处理异常数（status 非 CLOSED）|
|unassignedExceptionCount|integer(int64)|false|none||待指派异常数（status=PENDING 且 handler 为空）|
