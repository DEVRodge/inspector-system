安卓巡检端 UI 原型（重构流程版）

已按你的反馈重新组织为单线流程，去掉“暂存重传”和“地图”能力，聚焦巡检与异常闭环。

Material 3 风格 · FR-32~42 对齐 · 流程优先
主流程：登录 → 首页 → 任务列表 → 任务详情 → 设备清单 → 扫码 → 巡检表单提交。
异常分支（单独底部菜单）：发现异常自动生成记录 → 异常列表 → 异常详情（含照片）→ 处理异常（结果+照片）→ 确认处理（闭环）。
