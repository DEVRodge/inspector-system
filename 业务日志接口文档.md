# 业务日志

## GET 业务操作日志分页查询（仅巡检记录/异常任务写操作）

GET /inspection/biz-log/page

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|pageNumber|query|integer| 是 |页码|
|pageSize|query|integer| 是 |页面大小|
|bizType|query|string| 否 |业务类型：INSPECTION_RECORD/INSPECTION_EXCEPTION|
|bizId|query|integer(int64)| 否 |业务主键|
|operation|query|string| 否 |操作类型：SUBMIT/ASSIGN/PROCESS|
|success|query|boolean| 否 |是否成功：true/false|
|period.begin|query|string| 否 |开始时间|
|period.end|query|string| 否 |结束时间|
|keyword|query|string| 否 |关键字：操作描述/操作人|
